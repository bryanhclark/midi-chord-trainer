# midi-chord-trainer
A [Chord Trainer](https://bryanadams.io/midi-chord-trainer) built with:
- [Vite](https://vitejs.dev/)
- [Tone.js](https://tonejs.github.io/) 
- [Shadcn](https://ui.shadcn.com/).q

## install

```bash
npm install
```

## run 

```bash
npm run dev
```
